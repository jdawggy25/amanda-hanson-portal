## Project Overview

This documentation covers a complete rebuild of Amanda Hanson's coaching website, transitioning from WordPress/Elementor to a code-based ultra-premium solution while keeping Kajabi for course delivery.

### Key Objectives

1. **Performance**: Achieve 95+ Lighthouse score, sub-2s load times
2. **Conversion**: Optimize funnels for 40%+ conversion improvement
3. **SEO**: Rank page 1 for target keywords
4. **Tracking**: Full analytics with GA4, Clarity, Meta CAPI
5. **Scalability**: Support $1M+ annual marketing spend

## Implementation Timeline

### Phase 1: Foundation (Weeks 1-2)
- Set up Astro project with TypeScript
- Configure Cloudflare Pages deployment
- Set up Sanity CMS
- Implement base layout and navigation

### Phase 2: Core Development (Weeks 3-4)
- Build all core pages
- Integrate Kajabi API
- Set up email capture with Kit
- Implement Stripe for payments

### Phase 3: Optimization (Weeks 5-6)
- Image optimization (AVIF/WebP)
- Font subsetting and preloading
- Third-party script deferral
- Core Web Vitals validation

### Phase 4: Analytics & Testing (Weeks 7-8)
- GA4 configuration with custom events
- Microsoft Clarity setup
- Meta Pixel installation
- Cross-browser and mobile testing
- Launch
