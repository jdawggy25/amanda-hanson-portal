## Content Production Workflow

### Step 1: Assignment (Day 1)
- Assign to writer
- Share content brief
- Confirm deadline

### Step 2: First Draft (Day 2-4)
- Writer completes draft
- Self-edit for GEO compliance
- Submit for review

### Step 3: SEO Review (Day 5)
- Keyword optimization check
- Internal linking additions
- Schema markup verification

### Step 4: Clinical Review (Day 6) - If Required
- Medical accuracy check
- Terminology verification
- Compliance review

### Step 5: Final Edit (Day 7)
- Copy editing
- Formatting check
- Visual assets added

### Step 6: Publishing (Day 8)
- Upload to CMS
- Meta data entry
- Schema implementation
- Internal links added
- Social sharing scheduled

### Step 7: Post-Publish (Day 9+)
- Index verification
- Performance monitoring
- Promotion activities

## Content Resources

### Style Guide (Summary)

- **Voice**: Professional, authoritative, approachable
- **Tone**: Educational, helpful, not salesy
- **POV**: Second person (you) for guides
- **Citations**: Always cite sources
- **CTAs**: Soft sells, value-first

### Brand Keywords to Include

- V3 Biomedical
- Wound care platform
- Unified workflows
- Product access
- Compliance-driven
- Patient-focused

### Topics to Avoid

- Specific medical advice (liability)
- Competitor disparagement
- Unverified claims
- Political content
- Pricing specifics (unless approved)
