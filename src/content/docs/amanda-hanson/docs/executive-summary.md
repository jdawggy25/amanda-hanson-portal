## Technical Recommendation

### Recommended Stack

| Component | Solution | Rationale |
|-----------|----------|-----------|
| **Frontend** | Astro 4+ | Best performance, zero JS default |
| **Hosting** | Cloudflare Pages | Free, 310+ edge locations |
| **CMS** | Sanity | Real-time editing, excellent DX |
| **Images** | Cloudflare Images | Auto AVIF/WebP, $5/mo |
| **Email** | Kit (ConvertKit) | Creator-focused, automation |
| **Courses** | Kajabi (Keep) | Already in use, API integration |
| **Payments** | Stripe | Industry standard |
| **Analytics** | GA4 + Clarity | Full tracking + session replay |

### Why Astro Over Next.js

| Factor | Astro | Next.js |
|--------|-------|---------|
| Bundle Size | 15-50KB | 100-150KB |
| LCP | ~1.2s | ~1.8s |
| Learning Curve | Lower | Higher |
| Content Sites | Optimized | Good |
| Hosting Cost | $0 | $20+/mo |
